Dream Team - App

The Challenge:

"Dream Cricket Team" - Web App

Project Goal:

Build a single-page Angular application that allows a user to create and manage their ultimate 11-player Indian cricket squad. The application should be dynamic, user-friendly, and demonstrate a strong understanding of core Angular principles.

Core Features & Requirements:

  1. Create a Firestore Database to store "players' collection.
  2. Create a Firebase Cloud Function (REST API) to fetch data from Firestore DB. [ API GET : "/getPlayers" ]
  3. In Angular UI use HttpClient service to load all available Players from Firebase.
  4. Use Angular Routes for Home Page and About Page.
  5. Save application state in LocalStorage.
  6. Create separate components for different parts of the application.
  7. Use your own CSS files to give the application a professional and responsive look.
